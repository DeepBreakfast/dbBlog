# Full Stack JavaScript Blog Web App
* Front end: Vue.js + Element + Axios (requests)
* Back end: Express.js (router, controllers) + MongoDB/Mongoose (DB/ODM)
* Build tool: Parcel
* Testing: Jade

Features:
* Models:
  * User
  * Page/Post
    * Fields: Image, body,
  * Comment
* Control panel
* Single owner blog


Server is a simple REST API connected to MongoDB
